# 说明

>  非常简单的一个 vue + webpack4 的项目，适合作为入门练习。

>  如有问题，或者您发现问题并有非常好的解决方案，可以直接提交 Issues

## 项目运行（nodejs 7.0+）

``` bash
# 克隆到本地
git clone https://github.com/Gintoki-97/todo-list-v2.git

# 进入文件夹
cd todo-list-v2

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build
```
