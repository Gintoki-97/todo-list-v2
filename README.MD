## 项目运行（nodejs 7.0+）

``` bash
# 克隆到本地
git clone https://github.com/Gintoki-97/todo-list.git

# 进入文件夹
cd todo-list

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build
```
